# Google Antigravity Landing Page Clone

A modern **Vite + React** clone of the popular **Google Antigravity** landing page demo. This project replicates the playful anti-gravity effect shown in various Google experiments, where page elements float, drift, and respond to user interaction.

## 🚀 Features

- ⚡ Built with **Vite** for fast development and optimized builds
- ⚛️ **React** for component-based structure
- 🎈 Smooth anti-gravity animations using CSS and JavaScript
- 🖱️ Interactive floating elements that respond to cursor movement
- 📱 Fully responsive layout
- 🎨 Clean, minimal UI inspired by the original Google demo

## 🛠️ Tech Stack

- **Vite**
- **React**
- **TypeScript**
- **CSS / TailwindCSS**

## 🎮 How It Works

The landing page recreates Google’s antigravity effect by:

- Applying a floating force to UI elements
- Using requestAnimationFrame for smooth updates
- Tracking mouse position and applying subtle attraction/repulsion
- Adding random drift to mimic zero gravity

## 🧩 Components (example list)

- `FloatingLogo` – Google logo with anti-gravity animation
- `SearchBox` – Fake search bar with floating effect
- `Particles` – Optional particle drift system
- `Layout` – Page wrapper with centering and transitions
