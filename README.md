# Google Antigravity Landing Page Clone

A Next.js App Router clone of the Google Antigravity-inspired landing page with interactive particle motion, scroll inertia, and responsive sections.

## Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Three.js

## Run Locally

Install dependencies:

```bash
yarn install
```

Start development server:

```bash
yarn dev
```

Build for production:

```bash
yarn build
```

Start production server:

```bash
yarn start
```

## Project Structure

```text
app/
	globals.css
	layout.tsx
	page.tsx
components/
	common/
	layout/
	module/
lib/
	data/
	hooks/
	utils.ts
public/
```

## Notes

- Main route is implemented in `app/page.tsx`.
- Global styles are in `app/globals.css`.
- Interactive components are client components where required.
